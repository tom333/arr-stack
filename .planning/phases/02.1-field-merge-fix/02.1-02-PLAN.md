Three coordinated changes ship in lockstep:
1. **Apply-side merge** (`differ.py:merge_fields_for_put` + `sonarr.py:_execute` UPDATE branch wiring): for any FieldKV whose YAML value is `''`/`None`, take cluster's stored value (matched by name); else keep YAML value. `tags` is NOT merged (desired wins — the reconciler stamps `managed_tag_id`).
2. **Dump-side filter** (`dump.py`): drop FieldKV entries whose Sonarr-side value is `***REDACTED***` so the round-trip stays clean (D-36).
3. **Test suite extension**: respx unit tests in `test_differ.py` + integration tests in `test_reconcilers_sonarr.py` + round-trip test in `test_round_trip.py`. All gates (ruff, mypy, pytest, coverage ≥70%) green.

Why both apply AND dump: they form the two ends of the same contract (D-36 + D-31). Dump omits redacted entries → reload produces YAML where those fields are absent → reconciler's diff sees no entries → merge has nothing to preserve → round-trip stays at 0 actions. Without BOTH halves, the existing `test_round_trip_dump_apply_dry_run_is_noop` (canary) fails.

        def merge_fields_for_put[T: BaseModel](current: T, desired: T) -> dict[str, Any]:
            """Merge cluster's stored field values into desired body for PUT (D-31/D-32).

            For each entry in desired.fields[], if value is '' or None,
            take the corresponding cluster value (matched by `name`).
            Otherwise keep desired's value.

            ``tags`` is intentionally NOT merged — desired's tags list legitimately
            overrides cluster's (the reconciler appends managed_tag_id, D-02).
            """
            cur_dump = current.model_dump(exclude_none=True)
            des_dump = desired.model_dump(exclude_none=True, exclude=_READ_ONLY_FIELDS)
            cur_by_name = {f["name"]: f for f in cur_dump.get("fields", [])}
            merged_fields: list[dict[str, Any]] = []
            for des_f in des_dump.get("fields", []):
                v = des_f.get("value")
                if v == "" or v is None:
                    cur_f = cur_by_name.get(des_f["name"])
                    if cur_f is not None and cur_f.get("value") not in ("", None):
                        merged = dict(des_f)
                        merged["value"] = cur_f["value"]
                        log.info("merge_field_preserved", name=des_f["name"])
                        merged_fields.append(merged)
                        continue
                merged_fields.append(des_f)
            des_dump["fields"] = merged_fields
            return des_dump
