```python
class JellyfinClient(ArrApiClient):
    """Jellyfin 10.11.8 REST client — Phase 7 (D-07-AUTH-01).

    Diverges from ArrApiClient default in 3 ways:
    1. api_path = "" — Jellyfin uses bare /System/Info, /Library/VirtualFolders,
       etc. (no /api/v3 prefix). Setting api_path="" makes httpx.Client.base_url
       equal self.base_url exactly.
    2. auth_headers() returns the MediaBrowser format (D-07-AUTH-01 + Q9 probe).
       Registers arrconf as a distinct device in /Devices (auditable).
    3. Does NOT inherit from _ArrV3Client — Jellyfin has no forceSave mechanism
       (ADR-8 explicitly scopes forceSave to *arr v3 only ; spec.md §895).
    """

    api_path = ""  # Jellyfin endpoints live at /<resource>, not /api/v3/<resource>
    name = "jellyfin"

    def auth_headers(self) -> dict[str, str]:
        """MediaBrowser Token header — Jellyfin 10.11+ recommended (Q9 / D-07-AUTH-01)."""
        return {
            "Authorization": (
                f'MediaBrowser Token="{self.api_key}", '
                f'Client="arrconf", '
                f'Device="arrconf", '
                f'DeviceId="arrconf", '
                f'Version="0.5.0"'
            )
        }
```

```python
def _payloads_equivalent(current: dict[str, Any], desired: dict[str, Any]) -> bool:
    """Idempotence diff: True iff `current` matches `desired` on every key in `desired`.

    Extra keys in `current` (e.g. server-computed activeProfileName,
    server-managed timestamps) are IGNORED — we only care that the desired
    subset is already satisfied. Mirrors the Phase 5 D-05-MIG-01 idempotence
    pattern (SC#5 dispositive — second run = no-op).
    """
    return all(current.get(k) == v for k, v in desired.items())
```

```python
def _find_default_service(
    current: list[dict[str, Any]], resource_name: str
) -> dict[str, Any] | None:
    """Locate the single isDefault=true entry.

    Returns None if not bootstrapped (warning logged); raises ReconcileError if
    duplicated (malformed cluster state).
    """
    defaults = [s for s in current if s.get("isDefault") is True]
    if len(defaults) > 1:
        raise ReconcileError(
            f"seerr {resource_name}: cluster has {len(defaults)} isDefault=true entries — "
            f"expected exactly 1 per D-06-SCOPE-01 (ADR-7 single instance)"
        )
    if not defaults:
        log.warning(
            f"seerr_{resource_name}_no_default",
            hint=(...),
        )
        return None
    return defaults[0]
```

## Shared Patterns

### Idempotence diff helper
**Source:** `tools/arrconf/arrconf/reconcilers/seerr.py:97-105` `_payloads_equivalent`
**Apply to:** `_reconcile_libraries`, `_reconcile_users`, `_reconcile_server_config`, `_reconcile_plugins` in `reconcilers/jellyfin.py`.
```python
def _payloads_equivalent(current: dict[str, Any], desired: dict[str, Any]) -> bool:
    return all(current.get(k) == v for k, v in desired.items())
```
**Jellyfin delta:** For `_reconcile_server_config`, the diff is over the **7 allowlist keys only** (extra cluster keys ignored). For `PluginRepositories` (list-of-dicts), use set-by-URL comparison (Pitfall 7).

### Re-inject OpenAPI-required fields (D-06-CREDS-01 / D-06-OPENAPI-01 generalized)
**Source:** `tools/arrconf/arrconf/reconcilers/seerr.py:130-149` `_reconcile_settings_sonarr` apiKey + activeProfileName re-injection
**Apply to:** `_reconcile_users` in Jellyfin (re-inject `AuthenticationProviderId` + `PasswordResetProviderId` from cluster GET — Pitfall 6).
**Pattern:**
```python
desired_payload = desired_section.policy.model_dump()  # excludes the 2 required IDs
cluster_user = client.get(f"/Users/{user_id}")
cluster_policy = cluster_user["Policy"]
desired_payload["AuthenticationProviderId"] = cluster_policy["AuthenticationProviderId"]
desired_payload["PasswordResetProviderId"] = cluster_policy["PasswordResetProviderId"]
client._request("POST", f"/Users/{user_id}/Policy", json=desired_payload)
```

**Apply to:** Jellyfin branch in `__main__.py`. Avoids startup cost + circular import risk.

### Anti-leak grep before commit
**Source:** CLAUDE.md "Ne pas committer de secrets"; Phase 5/6 deviation #4 (snapshot.sh redaction defect)
**Apply to:** `snapshots/before-phase-7-*/`, `evidence/q9-put-probe.txt`, `tools/arrconf/tests/fixtures/jellyfin/*.json`.
**Patterns to grep:** `api_key=[a-f0-9]{32}`, `AccessToken`, `Bearer `, `Authorization:`, base64 with token-like length.

### dry_run signature contract
**Source:** `seerr.py:111` all `_reconcile_<X>(client, section, dry_run)` signatures
**Apply to:** All 4 Jellyfin `_reconcile_<X>` functions. When `dry_run=True`: emit `log.info("dry_run_skip", resource=<name>, ...)` and append `f"<resource>:dry_run:<key>"` to actions. Do NOT call POST.

### `_request` for non-id-suffix paths
**Source:** `seerr.py:162-166` and `seerr.py:313` use `client._request("POST", PATH, json=body)` directly (NOT `client.post()`) because the path is not the standard `{path}/{id}` shape, OR because 204 responses break `r.json()` on empty body.
**Apply to:** Jellyfin's `client._request("POST", "/Library/VirtualFolders/Paths", ...)`, `client._request("POST", "/System/Configuration", ...)`, `client._request("POST", f"/Users/{user_id}/Policy", ...)`, `client._request("POST", f"/Plugins/{id}/{version}/Enable", ...)`. All 4 endpoints return HTTP 204 — safer to use `_request` directly.
