## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser → FastAPI `/api/trash/*` | Untrusted `app` query param crosses here |
| FastAPI → disk (`trash-metadata/`) | Trusted read of committed assets only |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-27-05 | Tampering / path traversal | `app` query param → file path | mitigate | Enum allow-list gate (`app not in ("sonarr","radarr") → 400`) before any path build; filename is a fixed template; tested by `test_path_traversal_app_returns_400` |
| T-27-06 | Information disclosure / SSRF | metadata handler making outbound HTTP | mitigate | Handlers read disk only; SC#2 grep gate asserts no httpx/requests/urllib/github in handler bodies |
| T-27-07 | Spoofing / boundary breach | *arr URL constructed in arrconf-ui | mitigate (ADR-5) | grep gate for `8989/7878/9696/sonarr./radarr./prowlarr.` returns none across arrconf_ui/ |
| T-27-08 | Denial of service | huge catalog read on each request | accept | Assets ~500 KB, read+parse is sub-millisecond; LAN-trusted single-operator UI |
</threat_model>
