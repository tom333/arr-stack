<objective>
Restructure the Svelte 5 frontend from a two-tab (arrconf / configarr) state machine into a three-tab machine (intent.yml editable | arrconf.yml inspect | configarr.yml inspect). This plan delivers the SKELETON: API layer, types, constants, the two new display components (ReadOnlyInspector + MaterializationDiffPanel), HeaderBar 3rd tab + conditional save button, and the App.svelte rewire for tab switching, intent load, diff preview, and save. The intent FORM sections (profile editor, configarr raw editor, picker mount) come in plan 34-03.

Purpose: intent.yml becomes the only editable file; arrconf.yml + configarr.yml become read-only inspectors; the diff panel shows the exact generated bytes before commit (UI-04).
Output: 4 modified TS/Svelte files + 2 new Svelte components. Build + frontend tests green. No new npm packages.
</objective>
