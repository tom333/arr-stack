## Milestone Summary

**Key Decisions:**

- **D-01 Categories: separate generators module** — Pure functions `generate_for_app(cfg, app)` in `tools/arrconf/arrconf/generators/categories.py`, called from `__main__.py` before reconciler dispatch. Reconciler signatures unchanged.
- **D-02 Per-resource toggle (`merge_with_manual`)** — Manual flat-section non-empty → manual wins entirely; empty → Categories-derived. One-line predicate. Transition layer planned for deprecation in v0.4.0+.
- **D-03 5 tags/root_folders/DCs/RPMs per side** — Each Category produces exactly 1 resource per resource-type on Sonarr/Radarr. qBit categories named bare `<name>` (D-03a, not `<kind>-<name>`).
- **D-04 B2 allowlist comparator across 3 FP fixes** — Module-level `frozenset[str]` constants (`QBIT_CATEGORY_MANAGED_FIELDS`, `SEERR_USER_MANAGED_FIELDS`, `PROWLARR_APP_MANAGED_FIELDS` + `PROWLARR_APP_MANAGED_FIELD_NAMES` sub-field allowlist) mirror Jellyfin's `SERVER_CONFIG_ALLOWLIST` pattern. Filters cluster GET response BEFORE `model_validate()`.
- **D-05 Chart-pin co-bump pattern** — Every commit modifying `tools/arrconf/**` also stages `charts/arr-stack/values.yaml#arrconf.image.tag` in the SAME commit. Closes D-07-CHART-PIN-LOOP. Documented in CLAUDE.md + gsd-executor.md.
- **D-11-PLAN-STRUCTURE Omnibus 2 plans for Phase 11** — Vu la légèreté individuelle des items (1-2 tasks each), 11-A (4 REQs autonome) + 11-B (3 REQs operator-checkpoints).
- **D-11-RUFF-GATE Belt-and-suspenders** — `tests.yml` ruff format check + nouveau pre-commit hook (`.pre-commit-config.yaml` avec `astral-sh/ruff-pre-commit`).
- **D-11-REDACT-APPROACH Inline jq filter** — Réutilise le pattern existant de `tools/snapshot/README.md` Audit anti-leak Option A. 0 dépendance Python ajoutée.
- **Phase 8 ESO/Akeyless RETIRED 2026-05-22** — Bitnami sealed-secrets côté my-kluster est la solution long-terme. Aucune migration externe-secret planifiée. REQ-secret-management closed in spirit.
