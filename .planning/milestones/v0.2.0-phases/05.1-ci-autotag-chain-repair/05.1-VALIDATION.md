## Sampling Rate

- **After every task commit:** `helm lint charts/arr-stack/ -f examples/values-prod.yaml` is NOT relevant (no chart change). Instead: push PR branch, observe `chart-lint` workflow run.
- **After every plan wave:** Wave 1 = workflow YAML edits, validated by Wave 2 = post-merge gates (G-05.1-1/2/3).
- **Before `/gsd-verify-work`:** `:0.3.1` HEAD-probes 200/401 on GHCR (SC#4 dispositive) AND Renovate has opened the values.yaml bump PR (SC#5 dispositive).
- **Max feedback latency:** Pre-merge: ~2 min (CI lint). Post-merge: ~60 min (Renovate scan).
