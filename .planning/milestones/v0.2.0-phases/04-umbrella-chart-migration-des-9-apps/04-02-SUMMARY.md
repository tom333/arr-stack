## Decisions Made

- **OCI registry unavailable:** `oci://ghcr.io/bjw-s/helm/app-template:5.0.0` returns 404. Switched to `https://bjw-s-labs.github.io/helm-charts` which serves 5.0.0. This is a correction from the plan which specified the OCI path.
- **Helm 4 multi-alias deduplication:** With 10 aliases all using `name: app-template version: 5.0.0`, Helm 4 downloads 1 tgz but `helm template` requires the chart unpacked as a directory. Solution: `charts/*/charts/` stays gitignored (no tgz or unpacked dir committed), CI must run `helm dependency build` before `helm template`. ArgoCD runs this automatically.
- **Chart.lock committed:** Removed `charts/*/Chart.lock` from `.gitignore` (it was pre-added anticipating a "don't commit" choice). Chart.lock is committed for digest reproducibility.
- **configarr.yml !env tags:** The configarr config uses `!env VAR` custom YAML tags (configarr's env injection syntax). Python `yaml.safe_load` rejects these; `helm lint` accepts them (Helm renders the file as a string via `.Files.Get`). The file is valid for its consumer.
